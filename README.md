# Catch-up TV &amp; More repository add-on

This repo contains the repository add-on of Catch-up &amp; More in order to give the user the ability to keep Catch-up &amp; More automatically up to date.
There is different add-on for different Kodi version.

Leia users can use Krypton add-on.


## Tempo

### How to update Jarvis Release repository

```python
python create_repository.py -d ./zips/jarvis_release -i addons_xmls/jarvis_release/addons.xml -c addons_xmls/jarvis_release/addons.xml.md5 repo_addons_src/catchuptvandmore.kodi.jarvis.release-0.0.3/ repo_addons_src/catchuptvandmore.kodi.jarvis.release-0.0.4/ https://github.com/Catch-up-TV-and-More/plugin.video.catchuptvandmore\#kodi16
```
